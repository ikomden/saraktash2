Загрузка классификатора банков РФ

  • Выберите источник для загрузки классификатора банков:

    • с сайта агенства "РосбизнесКонсалтинг" (при работе в веб-клиенте возможна загрузка только с сайта РБК);

    • с диска "Информационно-технологическое сопровождение" - загрузка будет производиться из поставляемого на диске 1С:ИТС "Общероссийского классификатора банков" (ОКБ).

      • укажите Путь к диску ИТС.

  • Нажмите Загрузить.

  • После успешной загрузки нажмите Закрыть.

Загрузка банков может производиться автоматически, для этого необходимо настроить регламентное задание "Загрузка классификатора банков". При автоматической загрузке программа позволяет всегда поддерживать классификатор в актуальном состоянии.
